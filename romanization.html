한글 로마자 변환기 (일반명사, 표준발음, 영문이름, 영문주소, 학술용어)

문화체육관광부 고시 제2014-0042호에 따른 국어의 로마자 표기법을 적용하여 한글을 로마자로 변환합니다.

! 사용 시 유의사항
이 도구는 일반 명사/개별 단어 변환에 최적화되어 있습니다. 문장, 약어, 고유명사, 특수문자가 포함된 문구는 제한될 수 있습니다.
표준 발음법·로마자 표기법에는 예외 규정이 많아 100% 정확한 자동 변환은 불가능하므로 결과는 참고용으로만 활용해 주세요.
특히 조사(이/가, 을/를, 은/는 등)는 붙이지 말고 기본형으로 입력하는 것이 정확도에 도움이 됩니다.
모드별 활용 예시 How to use
일반 고유명사
지명·단어·브랜드명 등 일반 표기
예) 묵호 → Mukho
이름(성명)
여권·명함용 영문 성명 표기
예) 김민수 → Kim Minsu
영문주소
해외 배송용 영문 주소 변환(영문주소 API)
예) 서울특별시 종로구 세종대로 209, 3층
학술
한글 복원을 전제로 표기할 때
예) 값 → gabs
성은 대표 성씨 추천 표기를 우선 적용하고, 목록에 없는 성은 현행 표기법으로 변환합니다.
※ 브라우저 CORS 정책/승인키 상태에 따라 영문주소 API 호출이 실패할 수 있습니다. 실서비스는 서버 프록시 권장.
사용법: 입력 후 “변환” 버튼 또는 Ctrl+Enter로 변환합니다.
